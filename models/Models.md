## Model download

The provided analysis script will download the default model and labels upon first run.

You can download models manually from [Zenodo](https://zenodo.org/records/17631020).

We provide the following models:

- TorchScript model in FP32 format
- ONNX model in FP32 format

Please make sure to review the [license](../LICENSE) and [terms of use](../TERMS_OF_USE.md) before downloading the models.
